ARCHITECTURE:

Request → Pattern Filter (0-5ms)
             ↓ matched?
         ┌───YES → BLOCK (fast exit)
         └───NO → Bedrock API (100-300ms)
                      ↓ blocked?
                  ┌───YES → BLOCK
                  └───NO → ALLOW
